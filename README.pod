=head1 NAME

pzkcli - command line client for ZooKeeper

=head1 DESCRIPTION

An interactive command line client for ZooKeeper. Provides a shell like interface to ZooKeeper nodes.

Single commands can also be piped into STDIN.

=head1 SYNOPSIS

    pzkcli [SERVER]
